# Incentivo a Educação (ETEC)

Este projeto é um sistema web para gerenciamento de inscrições no programa de incentivo educacional da Cocal.

## Funcionalidades

- **Formulário Passo a Passo**: Cadastro intuitivo dividido em 3 etapas.
- **Integração com Google Sheets**: Os dados são salvos automaticamente em uma planilha do Google via Apps Script.
- **Upload de Arquivos**: Anexo de comprovantes de matrícula diretamente para o Google Drive.
- **Validação de Dados**: Verificação de campos obrigatórios e formatos corretos.
- **Design Responsivo**: Funciona bem em computadores e dispositivos móveis.

## Como Executar

Basta abrir o arquivo `index.html` em qualquer navegador moderno.

## Tecnologias Usadas

- HTML5
- CSS3 (Glassmorphism)
- JavaScript (Vanilla)
- Google Apps Script (Backend)
